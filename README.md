# Budget Planner AI MCP

Plan personal budgets using popular rules like 50/30/20 and track category allocations.

## Tools

- `calculate_50_30_20` — Split income into needs, wants, and savings.
- `plan_categories` — Plan a custom category budget.
- `check_overspend` — Compare planned vs actual spending.

## Install

```bash
pip install -e .
```

## Run

```bash
python -m budget-planner-ai-mcp.server
```
